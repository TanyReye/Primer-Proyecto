El sistema es una aplicación de consola desarrollada en C# que permite evaluar contenido digital como películas, series, documentales y eventos en vivo.
Su función principal es analizar si un contenido cumple con ciertas reglas técnicas y, con base en ello, decidir si puede ser publicado, rechazado o enviado a revisión.

El sistema también genera estadísticas en tiempo real sobre las evaluaciones realizadas, permitiendo al usuario tener un control general del proceso.


Abrir el proyecto en Visual Studio o el entorno de desarrollo utilizado.
Ejecutar el programa (tecla F5 o botón “Iniciar”).
Esperar a que aparezca el menú principal en la consola.
Seleccionar una opción ingresando el número correspondiente.
Seguir las instrucciones en pantalla para ingresar los datos solicitados.
Visualizar el resultado de la evaluación.
Repetir el proceso según sea necesario o salir del sistema.


Este proyecto consiste en el desarrollo de un sistema que simula una plataforma de control de contenido digital.
Su propósito es aplicar reglas específicas relacionadas con clasificación, duración, horario y nivel de producción para tomar decisiones automatizadas.

Además, incorpora un análisis de impacto que permite clasificar el contenido en diferentes niveles, ayudando a determinar si debe publicarse directamente o pasar a revisión.

El proyecto busca demostrar el uso de estructuras de control, validaciones y lógica de programación en un entorno práctico.
